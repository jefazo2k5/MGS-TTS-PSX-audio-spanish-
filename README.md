# MGS-TTS-PSX-audio-spanish-
Parche de inyección de audio castellano original PSX  a Metal Gear The Twin Snake PSX.


Proyecto de localización que busca reconstruir los diálogos en español (principalmente CODEC y cinemáticas) de Metal Gear Solid: The Twin Snakes (GameCube) utilizando como fuente los audios de Metal Gear Solid (PSX) en español, manteniendo el timing, offsets y estructura de los contenedores de audio del juego de GameCube.
